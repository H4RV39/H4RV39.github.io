🔧 First-time setup

MathsPoint needs a Firebase project to store shared data. This takes about 3 minutes and is completely free.

1 Go to console.firebase.google.com and create a new project (disable Google Analytics — you don't need it).

2 In your project: click Build → Firestore Database → Create database, choose Start in test mode, pick any location.

3 Go to Project settings → Your apps → Add app → Web (</>). Copy the config values below into the fields.

4 Go to Build → Authentication → Get started → Email/Password → Enable and save.

Config is saved in your browser. All your friends need to do this once too, or you can hardcode the values in the HTML.
